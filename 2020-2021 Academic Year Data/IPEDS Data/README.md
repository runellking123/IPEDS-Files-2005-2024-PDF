# IPEDS Data - 2020-2021 Academic Year

This folder contains official data from the National Center for Education Statistics (NCES) Integrated Postsecondary Education Data System (IPEDS).

## Files Included

### 1. IPEDS_Official_Summary.md
**Official IPEDS 2022 Data Feedback Report Summary**
- Degrees awarded (official count)
- Enrollment statistics
- Retention rates
- Graduation rates
- Financial aid information
- Peer comparisons

## Data Source

**Source:** U.S. Department of Education, National Center for Education Statistics
**Report:** IPEDS 2022 Data Feedback Report
**URL:** https://nces.ed.gov/ipeds/dfr/2022/ReportHTML.aspx?unitId=229887

**Data Year:** 2020-2021
**Retrieved:** December 16, 2025

## Official Use

This is **official federal education data** suitable for:
- External reporting and compliance
- Accreditation documentation
- Grant applications
- Public communications
- Peer institution benchmarking

## Comparison with Databricks Data

IPEDS data represents the official reported figures and may differ slightly from internal Databricks queries due to:
- Reporting period definitions
- Data collection methodologies
- Timing of data submission vs real-time database queries

See main README for reconciliation analysis.
