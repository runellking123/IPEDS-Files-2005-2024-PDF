# Data Source URLs

This folder contains reference links to external data sources for Wiley College institutional data.

## Files Included

### 1. NCES_IPEDS_Data_Feedback_Reports.txt
Links to official IPEDS Data Feedback Reports from the National Center for Education Statistics.
- Contains comprehensive annual institutional data
- Includes peer comparisons
- Years: 2020-2024 (covering 2019-20 through 2023-24 academic years)

### 2. NCES_College_Navigator.txt
Links to the NCES College Navigator profile for Wiley College.
- Real-time institutional data
- Searchable database
- Detailed sections for admissions, enrollment, retention, graduation, and financial aid

### 3. College_Comparison_Sites.txt
Links to third-party college comparison and information websites.
- College Tuition Compare: Trends and historical data
- College Factual: Academic life and outcome statistics

## Usage

These URLs provide access to:
- Official federal education data (IPEDS)
- Historical trend analysis
- Peer institution comparisons
- Student outcome metrics
- Financial information
